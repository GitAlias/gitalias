git merge-span-log

Find the commits that were introduced by a merge

Git alias:

merge-span-log = "!git log `git merge-span .. $1`"

Example:

git merge-span-log

Compare:

  • git merge-span (summary)
  • git merge-span-log (this alias)
  • git merge-span-diff
