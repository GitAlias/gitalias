# git bv

## List branch information with hash and subject

Git alias:

```git
bv = branch --verbose
```

Example:

```shell
git bv
```

When in list mode, then show the hash, the commit subject line, etc.

This is identical to doing `git b -v`.

Compare these:

* [git b](../git-b)

* [git bv](../git-bv) (this alias)

* [git bvv](../git-bvv)
