# git undo-commit-hard-clean

## Undo a commit with a hard reset and a clean

Git alias:

```git
undo-commit-hard-clean = !git reset --hard HEAD~1 && git clean -df
```

Example:

```shell
git undo-commit-hard-clean
```

See how to use this alias and related ones:

* [git undo-*](../git-undo)
* [git reset-*](../git-reset)
