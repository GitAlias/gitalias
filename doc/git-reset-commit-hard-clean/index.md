# git reset-commit-hard-clean

# Move backwards on the commit chain

Git alias:

```git
reset-commit-hard-clean = !git reset --hard HEAD~1 && git clean -df
```

Example:

```shell
git reset-commit-hard-clean
```

See how to use this alias and related ones:

* [git reset-*](../git-reset)
* [git undo-*](../git-undo)
